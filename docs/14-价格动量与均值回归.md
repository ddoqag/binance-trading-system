# 价格动量与均值回归因子

## 一、价格动量因子（Momentum）

动量是 **最稳定 Alpha**。

### 1. 20周期动量

```python
def momentum_20(df):
    return df["close"].pct_change(20)
```

---

### 2. 60周期动量

```python
def momentum_60(df):
    return df["close"].pct_change(60)
```

---

### 3. EMA趋势

```python
def ema_trend(df):
    ema_short = df["close"].ewm(span=10).mean()
    ema_long = df["close"].ewm(span=50).mean()
    return (ema_short - ema_long) / ema_long
```

---

### 4. MACD动量

```python
def macd_factor(df):
    ema12 = df["close"].ewm(span=12).mean()
    ema26 = df["close"].ewm(span=26).mean()
    return ema12 - ema26
```

---

### 5. 多周期动量组合

```python
def momentum_combined(df):
    mom5 = df["close"].pct_change(5)
    mom20 = df["close"].pct_change(20)
    mom60 = df["close"].pct_change(60)
    return 0.5 * mom5 + 0.3 * mom20 + 0.2 * mom60
```

## 二、均值回归因子（Mean Reversion）

短期市场有反转。

### 1. Z-score

```python
def zscore(df):
    mean = df["close"].rolling(50).mean()
    std = df["close"].rolling(50).std()
    return (df["close"] - mean) / std
```

**交易逻辑：**
```
z > 2   做空
z < -2  做多
```

---

### 2. Bollinger偏离

```python
def bollinger(df):
    ma = df["close"].rolling(20).mean()
    std = df["close"].rolling(20).std()
    return (df["close"] - ma) / std
```

**策略：**
```
触及上轨做空
触及下轨做多
```

---

### 3. RSI 因子

```python
def rsi_factor(df, window=14):
    delta = df["close"].diff()
    gain = (delta.where(delta > 0, 0)).rolling(window=window).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window=window).mean()
    rs = gain / loss
    return 100 - (100 / (1 + rs))
```

---

### 4. 短期反转因子

```python
def short_reversal(df):
    return -df["close"].pct_change(5)
```

**逻辑：**
```
涨多了就跌
跌多了就涨
```

## 三、构建因子计算框架

### 推荐结构

```
factor-engine/
├── factors/
│   ├── momentum.py
│   ├── volatility.py
│   ├── volume.py
│   └── orderflow.py
└── engine.py
```

---

### 核心执行器

```python
import pandas as pd
from factors import momentum, volatility

class FactorEngine:

    def __init__(self):
        self.factors = {
            "momentum": momentum.momentum_20,
            "volatility": volatility.realized_vol,
            "zscore": momentum.zscore
        }

    def compute(self, df):
        result = pd.DataFrame(index=df.index)
        for name, func in self.factors.items():
            result[name] = func(df)
        return result
```

**输出：**
```
time | momentum | volatility | zscore
```

## 四、因子 IC 测试（最重要）

**IC = Information Coefficient**

衡量：**因子 vs 未来收益**

### 计算方法

```python
def compute_ic(factor, future_return):
    return factor.corr(future_return)
```

---

### 完整示例

```python
df["future_return"] = df["close"].pct_change(10).shift(-10)

ic = compute_ic(
    df["momentum"],
    df["future_return"]
)

print("IC:", ic)
```

---

### IC 解释

| IC | 含义 |
|----|------|
| >0.1 | 很强 |
| 0.05 | 有效 |
| <0.02 | 噪音 |

---

## 五、IC 稳定性测试

不仅要 IC 大，还要 **稳定**。

```python
def rolling_ic(factor, future_return):
    return factor.rolling(200).corr(future_return)
```

**指标：**
- IC mean
- IC std
- IC IR

**IR = IC_mean / IC_std**

**机构标准：IR > 0.5**

### 方法2：相关性过滤

避免重复因子：

```python
corr = factor_df.corr()

drop_list = []

for i in corr.columns:
    for j in corr.columns:
        if abs(corr.loc[i,j]) > 0.8:
            drop_list.append(j)
```
