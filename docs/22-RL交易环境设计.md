# RL 交易环境设计

## 一、RL交易系统整体架构

完整结构：

```
Market Data
     ↓
Feature Engine (100+ factors)
     ↓
State Vector
     ↓
RL Trading Environment
     ↓
RL Agent (PPO / SAC)
     ↓
Action
     ↓
Execution Engine
     ↓
Reward (PnL / Risk)
     ↓
Policy Update
```

## 四、Action（动作空间）

常见三种设计：

### 1. 离散动作

```
0 HOLD
1 BUY
2 SELL
```

**优点：** 训练稳定

**缺点：** 不够精细

---

### 2. 连续仓位

```
action ∈ [-1, 1]
```

**解释：**
- `-1` = full short (全空)
- `0` = flat (空仓)
- `1` = full long (全多)

**这是机构常用方式。**

---

### 3. 混合动作

```
[action_type, size]
```

- `action_type`: BUY/SELL/HOLD
- `size`: 0~1

## 五、Reward Function（奖励函数设计）

### 1. 简单 PnL 奖励

```python
def compute_reward(portfolio_value, prev_value):
    return portfolio_value - prev_value
```

---

### 2. 风险调整奖励

```python
def reward_with_risk(pnl, volatility, lambda_risk=0.1):
    return pnl - lambda_risk * volatility
```

---

### 3. Sharpe 奖励

```python
def sharpe_reward(returns, window=30):
    return returns.mean() / (returns.std() + 1e-6)
```

---

### 4. 完整奖励函数

```python
class TradingReward:

    def __init__(self, risk_aversion=0.1):
        self.risk_aversion = risk_aversion
        self.prev_value = 10000.0

    def compute(self, portfolio, action):
        # PnL component
        pnl = portfolio.value - self.prev_value

        # Risk penalty
        volatility = portfolio.returns_std()
        risk_penalty = self.risk_aversion * volatility

        # Transaction cost penalty
        turnover = abs(action - portfolio.position)
        cost_penalty = 0.001 * turnover

        # Drawdown penalty
        drawdown = portfolio.max_drawdown()
        dd_penalty = 0.01 * drawdown

        reward = pnl - risk_penalty - cost_penalty - dd_penalty

        self.prev_value = portfolio.value

        return reward
```

## 六、Trading Environment（交易环境）

### Gym 风格实现

```python
import gym
from gym import spaces
import numpy as np

class TradingEnv(gym.Env):

    def __init__(self, data, initial_capital=10000):
        super().__init__()

        self.data = data
        self.initial_capital = initial_capital

        # Action space: continuous position [-1, 1]
        self.action_space = spaces.Box(
            low=-1, high=1, shape=(1,), dtype=np.float32
        )

        # Observation space: state vector
        self.observation_space = spaces.Box(
            low=-np.inf, high=np.inf, shape=(20,), dtype=np.float32
        )

        self.reset()

    def reset(self):
        """Reset environment to start state"""
        self.current_step = 0
        self.cash = self.initial_capital
        self.position = 0.0
        self.asset_held = 0.0
        self.trades = []

        return self._get_state()

    def step(self, action):
        """Execute one step in environment"""

        # Get current market data
        current_price = self.data["close"].iloc[self.current_step]

        # Action is target position [-1, 1]
        target_position = float(action[0])
        position_diff = target_position - self.position

        # Calculate trade size
        trade_size = position_diff * self.cash / current_price

        # Execute trade (with fees)
        fee = abs(trade_size) * current_price * 0.001
        self.cash -= fee

        # Update position
        self.position = target_position
        self.asset_held = trade_size

        # Update portfolio value
        portfolio_value = self.cash + self.asset_held * current_price

        # Compute reward
        reward = self._compute_reward(portfolio_value)

        # Move to next step
        self.current_step += 1
        done = self.current_step >= len(self.data) - 1

        # Info dict
        info = {
            "portfolio_value": portfolio_value,
            "position": self.position,
            "trades": len(self.trades)
        }

        return self._get_state(), reward, done, info

    def _get_state(self):
        """Return current state observation"""
        # Get feature data
        features = self.data.iloc[self.current_step]

        state = np.array([
            features["momentum"],
            features["volatility"],
            features["volume_spike"],
            features["rsi"],
            self.position,
            self.cash / self.initial_capital,
            # ... more features
        ])

        return state.astype(np.float32)

    def _compute_reward(self, portfolio_value):
        """Compute reward signal"""
        # Simple return-based reward
        prev_value = getattr(self, "prev_value", self.initial_capital)
        reward = (portfolio_value - prev_value) / self.initial_capital
        self.prev_value = portfolio_value
        return reward
```
