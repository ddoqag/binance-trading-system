# 交易执行优化（Execution Optimization）

## 一、执行问题（Execution Problem）

假设策略需要：**买入 100 BTC**

如果直接市价买入：**价格可能被推高**

### 示例

```
100 BTC 市价单
     ↓
吃掉多个订单簿层级
     ↓
平均成交价格变差
```

这就是 **Market Impact（市场冲击）**。

---

## 二、执行优化目标

执行算法的目标通常是：**最小化交易成本**

### 成本包括

| 成本类型 | 说明 |
|---------|------|
| Market Impact | 市场冲击 |
| Spread | 买卖价差 |
| Fees | 手续费 |
| Slippage | 滑点 |

### 总成本公式

```
Execution Cost = Impact + Spread + Fees + Slippage
```

## 四、TWAP（Time Weighted Average Price）

TWAP 是最简单的执行算法。

### 核心思想

**把订单平均分配到时间上**

### 示例

- 订单：100 BTC
- 时间：1 小时完成

**拆分：** 每分钟买入 1.67 BTC

### 代码示例

```python
def twap_execution(total_size, duration_minutes, intervals):
    """TWAP 执行策略"""

    interval_minutes = duration_minutes / intervals
    slice_size = total_size / intervals

    schedule = []
    for i in range(intervals):
        schedule.append({
            "time": i * interval_minutes,
            "size": slice_size
        })

    return schedule

# 使用示例
schedule = twap_execution(
    total_size=100,
    duration_minutes=60,
    intervals=60
)
```

### 优点

- 简单
- 稳定

### 缺点

- 忽略市场流动性

## 五、VWAP（Volume Weighted Average Price）

VWAP 根据 **市场成交量** 调整交易速度。

### 公式

```
VWAP = Σ(price × volume) / Σ(volume)
```

### 执行策略

| 市场情况 | 交易策略 |
|---------|---------|
| 成交量大 | 多交易 |
| 成交量小 | 少交易 |

### 代码示例

```python
def vwap_execution(total_size, market_volume, participation_rate=0.1):
    """VWAP 执行策略"""

    # 目标成交量 = 市场成交量 × 参与率
    target_volume = market_volume * participation_rate

    # 计算本次下单大小
    order_size = min(target_volume, total_size)

    return order_size
```

### 优点

**更贴近市场交易节奏**

## 六、POV（Participation of Volume）

POV 表示：**参与市场成交量的比例**

### 示例

```
POV = 5%
```

意味着：
- 市场成交 100 BTC
- 你交易 5 BTC

### 代码示例

```python
def pov_execution(market_volume, pov_rate=0.05):
    """POV 执行策略"""

    order_size = market_volume * pov_rate
    return order_size
```

### 优点

**自动适应市场流动性**

---

## 七、Iceberg 订单（冰山订单）

Iceberg（冰山订单）隐藏真实订单大小。

### 示例

**真实订单：100 BTC**

**订单簿只显示：10 BTC**

成交后继续挂单：再显示 10 BTC

### 优势

**避免暴露交易意图**

---

## 八、Sniper 算法

通常用于：**套利策略**

## 十二、动态执行策略

执行策略可以动态调整。

### 示例 1

**如果：市场波动 ↑**

**策略：减少交易**

### 示例 2

**如果：流动性 ↑**

**策略：加快交易**

### 代码示例

```python
def dynamic_execution(volatility, liquidity, base_size):
    """动态执行策略"""

    if volatility > threshold_vol:
        # 波动大，减少交易
        return base_size * 0.5
    elif liquidity > threshold_liq:
        # 流动性好，加快交易
        return base_size * 1.5
    else:
        return base_size
```
