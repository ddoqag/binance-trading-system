# 实时风险控制系统（Real-time Risk Engine）

## 一、风险系统的目标

在真实交易环境中，**风险控制的重要性甚至高于策略本身**。

许多机构（例如 Goldman Sachs、Citadel、Two Sigma）都会把风险系统独立出来，作为 **交易系统最优先级的模块**。

### 核心原则

```
策略可以亏钱
系统绝不能失控
```

### 实时风险系统需要解决四个核心问题

| 问题 | 说明 |
|------|------|
| 限制最大损失 | Limit max loss |
| 防止异常交易 | Prevent abnormal trading |
| 控制杠杆风险 | Control leverage risk |
| 保护系统稳定 | Protect system stability |

换句话说：

```
Risk Engine = Trading Firewall
```

## 四、策略级风险控制

每个策略必须设置限制：

| 限制类型 | 推荐值 |
|---------|--------|
| 最大仓位 | 5 BTC |
| 最大单日亏损 | 2% |
| 最大交易频率 | 50 orders/min |

### 代码示例

```python
class StrategyRiskLimit:

    def __init__(self, max_position=5.0, max_daily_loss=0.02):
        self.max_position = max_position
        self.max_daily_loss = max_daily_loss

    def check_position(self, current_position):
        """检查仓位"""
        if abs(current_position) > self.max_position:
            return False
        return True

    def check_order(self, position, order_size):
        """检查订单"""
        new_position = position + order_size
        return self.check_position(new_position)
```

## 五、账户级风险控制

### 账户层面限制

| 限制类型 | 推荐值 |
|---------|--------|
| 总仓位 | < 50% Equity |
| 杠杆比例 | < 3x |
| 保证金水平 | 安全阈值 |

### 代码示例

```python
class AccountRiskManager:

    def __init__(self, max_leverage=3.0, max_exposure=0.5):
        self.max_leverage = max_leverage
        self.max_exposure = max_exposure

    def check_leverage(self, equity, position_value):
        """检查杠杆"""
        leverage = position_value / equity
        return leverage <= self.max_leverage

    def check_exposure(self, equity, total_position):
        """检查总暴露"""
        exposure = total_position / equity
        return exposure <= self.max_exposure
```

## 六、回撤保护（Drawdown Protection）

系统需要监控 **实时回撤**。

### 示例

```
Max Drawdown = 10%
```

如果超过：**自动停止交易**

### 代码

```python
class DrawdownProtection:

    def __init__(self, max_drawdown=0.1):
        self.max_drawdown = max_drawdown
        self.peak_equity = 0.0

    def update(self, current_equity):
        """更新峰值"""
        if current_equity > self.peak_equity:
            self.peak_equity = current_equity

    def check(self, current_equity):
        """检查回撤"""
        drawdown = (self.peak_equity - current_equity) / self.peak_equity
        return drawdown <= self.max_drawdown

    def should_stop(self, current_equity):
        """是否应该停止交易"""
        return not self.check(current_equity)
```

## 七、仓位限制（Position Limits）

### 限制单资产仓位

| 资产 | 最大仓位 |
|------|---------|
| BTC | ≤ 10% |
| ETH | ≤ 10% |
| Altcoins | ≤ 20% |

这样可以防止：**单资产风险过大**

### 代码示例

```python
class PositionLimits:

    def __init__(self):
        self.limits = {
            "BTCUSDT": 0.1,   # 10%
            "ETHUSDT": 0.1,   # 10%
            "BNBUSDT": 0.05,  # 5%
            "SOLUSDT": 0.05,  # 5%
            "XRPUSDT": 0.05   # 5%
        }

    def check_position(self, symbol, position_value, equity):
        """检查单资产仓位"""
        limit = self.limits.get(symbol, 0.1)
        position_pct = position_value / equity
        return position_pct <= limit
```

## 八、订单频率控制

防止系统 bug 导致：**无限下单**

### 限制

```
Orders per second
```

### 代码

```python
from collections import deque
import time

class OrderRateLimiter:

    def __init__(self, max_orders_per_minute=100):
        self.max_orders = max_orders_per_minute
        self.order_times = deque()

    def check(self):
        """检查是否可以下单"""
        now = time.time()

        # 移除1分钟前的记录
        while self.order_times and now - self.order_times[0] > 60:
            self.order_times.popleft()

        if len(self.order_times) >= self.max_orders:
            return False

        return True

    def record_order(self):
        """记录下单"""
        self.order_times.append(time.time())
```

## 九、异常价格检测

有时交易所会出现：**错误价格**

### 示例

```
BTC = 1 USDT
```

系统必须检测异常。

### 代码

```python
class PriceAnomalyDetector:

    def __init__(self, max_deviation=0.2):
        self.max_deviation = max_deviation
        self.last_valid_price = None

    def check(self, current_price, market_price=None):
        """检查价格是否异常"""

        if market_price is not None:
            deviation = abs(current_price - market_price) / market_price
            return deviation <= self.max_deviation

        if self.last_valid_price is not None:
            deviation = abs(current_price - self.last_valid_price) / self.last_valid_price
            if deviation <= self.max_deviation:
                self.last_valid_price = current_price
                return True
            return False

        self.last_valid_price = current_price
        return True
```

## 十、滑点保护

如果订单价格偏离太大：**禁止交易**

### 示例

```
Max Slippage = 0.5%
```

### 代码

```python
class SlippageProtection:

    def __init__(self, max_slippage=0.005):
        self.max_slippage = max_slippage

    def check(self, expected_price, actual_price):
        """检查滑点"""
        slippage = abs(actual_price - expected_price) / expected_price
        return slippage <= self.max_slippage

    def should_cancel(self, expected_price, actual_price):
        """是否应该取消订单"""
        return not self.check(expected_price, actual_price)
```

## 十一、流动性保护

如果市场流动性不足：**禁止大额交易**

### 代码

```python
class LiquidityProtection:

    def __init__(self, max_participation=0.1):
        self.max_participation = max_participation

    def check_order_size(self, order_size, orderbook_depth):
        """检查订单大小是否合适"""
        return order_size <= orderbook_depth * self.max_participation

    def reduce_order(self, order_size, orderbook_depth):
        """减少订单大小以适应流动性"""
        max_size = orderbook_depth * self.max_participation
        return min(order_size, max_size)
```

## 十二、系统级熔断机制

当系统异常时：**Kill Switch**

### 触发场景

```
网络延迟过高
交易所 API 异常
PnL 快速下降
```

系统自动：**停止所有交易**

### 代码示例

```python
class KillSwitch:

    def __init__(self):
        self.triggered = False
        self.reasons = []

    def trigger(self, reason):
        """触发熔断"""
        self.triggered = True
        self.reasons.append(reason)

    def reset(self):
        """重置"""
        self.triggered = False
        self.reasons = []

    def check_latency(self, latency, threshold=5.0):
        """检查延迟"""
        if latency > threshold:
            self.trigger(f"High latency: {latency}s")
        return not self.triggered

    def check_pnl_drop(self, pnl_drop, threshold=0.05):
        """检查 PnL 快速下降"""
        if pnl_drop < -threshold:
            self.trigger(f"PnL dropped: {pnl_drop}")
        return not self.triggered
```

## 十三、自动减仓系统

风险系统可以自动：**降低仓位**

### 示例

**如果：波动率 ↑**

**系统：减少 50% 仓位**

### 代码

```python
class AutoPositionReducer:

    def __init__(self, volatility_threshold=0.02):
        self.volatility_threshold = volatility_threshold

    def should_reduce(self, volatility):
        """是否应该减仓"""
        return volatility > self.volatility_threshold

    def get_reduction_factor(self, volatility):
        """获取减仓比例"""
        if volatility > self.volatility_threshold * 2:
            return 0.5  # 减仓 50%
        elif volatility > self.volatility_threshold:
            return 0.3  # 减仓 30%
        else:
            return 0.0  # 不减仓
```

## 二十一、综合风险引擎

### 完整示例

```python
class ComprehensiveRiskEngine:
    """综合风险引擎"""

    def __init__(self):
        self.position_limits = PositionLimits()
        self.drawdown_protection = DrawdownProtection()
        self.order_rate_limiter = OrderRateLimiter()
        self.price_anomaly = PriceAnomalyDetector()
        self.slippage_protection = SlippageProtection()
        self.kill_switch = KillSwitch()

    def validate_order(self, order, market_data, portfolio):
        """验证订单"""

        # 1. 检查熔断
        if self.kill_switch.triggered:
            return False, "Kill switch triggered"

        # 2. 检查订单频率
        if not self.order_rate_limiter.check():
            return False, "Order rate limit exceeded"

        # 3. 检查价格异常
        if not self.price_anomaly.check(order.price, market_data["price"]):
            return False, "Price anomaly detected"

        # 4. 检查仓位限制
        new_position = portfolio.position + order.size
        if not self.position_limits.check_position(
            order.symbol,
            new_position * order.price,
            portfolio.equity
        ):
            return False, "Position limit exceeded"

        # 5. 检查回撤
        if self.drawdown_protection.should_stop(portfolio.equity):
            return False, "Max drawdown exceeded"

        # 6. 检查滑点
        if not self.slippage_protection.check(
            order.expected_price,
            order.price
        ):
            return False, "Slippage too high"

        # 所有检查通过
        self.order_rate_limiter.record_order()
        return True, "OK"
```
