# 交易环境与状态空间

## 一、构建交易环境（Trading Environment）

强化学习必须有 **环境（Environment）**。

### 环境模拟

| 要素 | 说明 |
|------|------|
| 市场价格 | Market Price |
| 订单执行 | Order Execution |
| 手续费 | Trading Fees |
| 滑点 | Slippage |

很多团队会使用 **OpenAI Gym 风格环境**。

---

## 二、环境接口

```python
class TradingEnv:

    def reset(self):
        """开始新的交易周期"""
        pass

    def step(self, action):
        """执行交易，返回奖励"""
        pass
```

### `reset()`

```
开始新的交易周期
```

### `step()`

```
执行交易
返回奖励
```

## 三、状态空间（State）

State 是模型看到的市场信息。

### 常见 State 包括

| 信息类型 | 说明 |
|---------|------|
| 价格 | Price |
| 成交量 | Volume |
| 技术指标 | Technical Indicators |
| 订单簿 | Order Book |
| 资金费率 | Funding Rate |

---

### 示例

```python

    price,
    volume,
    volatility,
    momentum,

]
```

也可以使用：**最近 N 根 K 线**

例如：**过去 50 个时间步**

## 四、动作空间（Action）

Action 是模型可以做的决策。

### 常见设计

#### 离散动作

```
0 = Hold
1 = Buy
2 = Sell
```

**示例：**
```python
action_space = 3
```

---

#### 连续动作（更高级的设计）

范围：`-1 ~ 1`

表示：**仓位比例**

| 值 | 含义 |
|----|------|
| -1 | 全空 |
| 0 | 空仓 |
| 1 | 全多 |

---

## 五、奖励函数（Reward）

Reward 是强化学习最关键部分。

### 简单奖励

```python
reward = portfolio_value_t - portfolio_value_t_minus_1
```

---

### 更真实的 Reward

加入交易成本：

```python
reward = pnl - trading_cost
```

其中：
```
trading_cost = fee + slippage
```

---

### 风险调整奖励（Risk-adjusted Reward）

例如：**Sharpe Ratio**

```python
reward = return / volatility
```

这样可以：**避免高风险策略**

## 十三、AI 交易模型难点
