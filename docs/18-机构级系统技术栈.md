## 五、行情采集服务

`market-data-service/binance_ws.py`

```python
import json
import websocket
from kafka import KafkaProducer

producer = KafkaProducer(
    bootstrap_servers="kafka:9092",
    value_serializer=lambda v: json.dumps(v).encode()
)

def on_message(ws, message):
    data = json.loads(message)
    tick = {
        "symbol": data["s"],
        "price": float(data["p"]),
        "volume": float(data["q"])
    }
    producer.send("market_ticks", tick)

ws = websocket.WebSocketApp(
    "wss://stream.binance.com:9443/ws/btcusdt@trade",
    on_message=on_message
)

ws.run_forever()
```

**数据流：**
```
Binance WS
   ↓
Kafka topic: market_ticks
```

## 七、Ray 分布式策略

策略可以并行运行：

```python
import ray

ray.init()

@ray.remote
class Strategy:
    def run(self):
        print("running strategy")

workers = [Strategy.remote() for _ in range(10)]

for w in workers:
    ray.get(w.run.remote())
```

**支持：100+策略同时运行**

## 九、机构级风控系统

风险层通常包含：

```
Risk Layers

1 单笔订单限制
2 单策略仓位限制
3 账户总风险
4 相关性风险
5 VaR
6 最大回撤
```

**Go 示例：**
```go
type RiskEngine struct {
    MaxOrder float64
    MaxPosition float64
}

func (r *RiskEngine) Check(price float64, qty float64) bool {
    value := price * qty
    if value > r.MaxOrder {
        return false
    }
    return true
}
```
