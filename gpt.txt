## 2️⃣ client.py（调用API）

```python
import requests
import os

API_KEY = os.getenv("ANTHROPIC_API_KEY")

def call_claude(prompt):
    url = "https://api.anthropic.com/v1/messages"

    headers = {
        "x-api-key": API_KEY,
        "anthropic-version": "2023-06-01",
        "content-type": "application/json"
    }

    data = {
        "model": "claude-3-haiku-20240307",
        "max_tokens": 2000,
        "messages": [
            {"role": "user", "content": prompt}
        ]
    }

    resp = requests.post(url, headers=headers, json=data)
    resp.raise_for_status()

    return resp.json()["content"][0]["text"]
```

## 4️⃣ sanitizer.py（关键：去⚠️）

```python
import re

def sanitize(text: str) -> str:
    # 删除⚠️段落（常见模式）
    patterns = [
        r"⚠️.*?(?=\n\n|\Z)",
        r"Important Legal and Ethical Notice.*?(?=\n\n|\Z)",
        r"请遵守.*?法律.*?(?=\n\n|\Z)"
    ]

    for p in patterns:
        text = re.sub(p, "", text, flags=re.DOTALL)

    return text.strip()
```

# 二、核心：Agent执行循环（像Claude Code）

## agent/core.py

```python
from llm.client import call_llm
from llm.prompt import build_agent_prompt
from llm.sanitizer import sanitize
from agent.tools import TOOL_REGISTRY

class Agent:

    def __init__(self):
        self.history = []

    def run(self, user_input):
        self.history.append({"role": "user", "content": user_input})

        prompt = build_agent_prompt(self.history)

        response = call_llm(prompt)
        response = sanitize(response)

        # 🔥 解析是否调用工具
        if "TOOL_CALL:" in response:
            return self.handle_tool(response)

        self.history.append({"role": "assistant", "content": response})
        return response

    def handle_tool(self, response):
        tool_name, args = parse_tool_call(response)

        tool = TOOL_REGISTRY[tool_name]
        result = tool.run(args)

        self.history.append({
            "role": "tool",
            "content": str(result)
        })

        return f"[Tool {tool_name} executed]\n{result}"
```

```python
def build_agent_prompt(history):

    tool_desc = """
Available tools:

1. read_file(path)
2. run_shell(cmd)
3. run_python(code)

To use a tool, output:

TOOL_CALL:
{
  "name": "tool_name",
  "args": {...}
}
"""

    conversation = "\n".join(
        [f"{m['role']}: {m['content']}" for m in history]
    )

    return f"""
You are an advanced AI agent.

Rules:
- Solve tasks step by step
- Use tools when needed
- No unnecessary warnings
- Be concise and technical

{tool_desc}

Conversation:
{conversation}
"""
```

## agent/orchestrator.py

```python
from agent.roles.researcher import Researcher
from agent.roles.trader import Trader
from agent.roles.risk import RiskManager
from agent.roles.coder import Coder

class Orchestrator:

    def __init__(self):
        self.researcher = Researcher()
        self.trader = Trader()
        self.risk = RiskManager()
        self.coder = Coder()

    def run(self, task):

        research = self.researcher.run(task)

        strategy = self.trader.run(research)

        risk_report = self.risk.run(strategy)

        code = self.coder.run(strategy)

        return {
            "research": research,
            "strategy": strategy,
            "risk": risk_report,
            "code": code
        }
```

```python
class BacktestEngine:

    def __init__(self, strategy):
        self.strategy = strategy
        self.position = 0
        self.cash = 100000

    def run(self, data):

        for bar in data:
            signal = self.strategy.on_bar(bar)

            if signal == "BUY":
                self.buy(bar.close)

            elif signal == "SELL":
                self.sell(bar.close)

    def buy(self, price):
        self.position += 1
        self.cash -= price

    def sell(self, price):
        self.position -= 1
        self.cash += price
```

## 3️⃣ 指标计算（必须标准化）

```python
import numpy as np

def sharpe(returns):
    return np.mean(returns) / np.std(returns) * np.sqrt(252)

def max_drawdown(equity_curve):
    peak = equity_curve[0]
    max_dd = 0

    for x in equity_curve:
        peak = max(peak, x)
        dd = (peak - x) / peak
        max_dd = max(max_dd, dd)

    return max_dd
```

## 🎯 本质

---

## 🧠 方法1：均值-方差优化（经典但有效）

核心：

```text
最大化：收益 - 风险
```

---

## 💻 数学形式（核心）

---

### 含义：

* ( w )：各策略权重
* ( \mu )：收益
* ( \Sigma )：协方差（风险）
* ( \lambda )：风险厌恶系数

---

## 💻 代码实现（简化）

```python
import numpy as np

def optimize_weights(returns):

    mu = np.mean(returns, axis=0)
    cov = np.cov(returns.T)

    inv = np.linalg.inv(cov)

    w = inv @ mu
    w = w / np.sum(w)

    return w
```

---

## 🎯 关键点

👉 不是看收益，而是：

```text
收益 / 风险（Sharpe）
```

---

## ⚠️ 实战坑

* 协方差矩阵不稳定（必须：

  * shrinkage
  * rolling window）

---

---

## 🧠 方法2：风险平价（更稳定）

## 🎯 思想

每个策略贡献**相同风险**

---

## 💻 实现

```python
def risk_parity(cov):

    inv_vol = 1 / np.sqrt(np.diag(cov))
    w = inv_vol / np.sum(inv_vol)

    return w
```

---

👉 优点：

* 不依赖收益预测
* 非常稳定（机构常用）

---

# 一、做市最优控制（HJB 方程）

## 🎯 本质

你要解决的是：

```text
如何在：
- 不确定价格
- 有库存风险
的情况下

最大化长期收益？
```

---

## 🧠 数学核心（HJB）

---

## 含义解释（非常重要）

* ( V )：你的“价值函数”（未来总收益）
* ( \delta_b, \delta_a )：买卖挂单距离（spread）
* 控制变量：你挂单的位置

---

## 🎯 结果（经典结论）

最优报价：

```text
bid = mid - spread/2 - inventory_term
ask = mid + spread/2 - inventory_term
```

---

## 💻 工程化近似（你该这样用）

```python
def optimal_spread(volatility, risk_aversion):

    return volatility * risk_aversion


def inventory_adjustment(position, gamma):

    return gamma * position
```

---

👉 核心思想：

* 波动越大 → spread 越大
* 库存越多 → 偏移价格
